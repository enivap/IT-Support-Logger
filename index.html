เชื่อมต่อ Google Sheets

วาง URL ของ Google Apps Script Web App ที่นี่ ข้อมูลทุก case จะ sync ไปเก็บใน Google Sheet ของคุณอัตโนมัติ

วิธีตั้งค่า Apps Script (ทำครั้งเดียว)
1. เปิด sheets.new สร้าง Google Sheet ใหม่
2. ไปที่ Extensions → Apps Script
3. ลบโค้ดเก่าออก แล้ว copy โค้ดจากไฟล์ Code.gs วางลงไป
4. กด Deploy → New deployment → Web App
5. ตั้ง "Who has access" เป็น Anyone → Deploy
6. Copy URL ที่ได้ มาวางในช่องด้านบน
